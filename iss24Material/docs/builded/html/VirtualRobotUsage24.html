VirtualRobotUsage24¶

Il sonar su RaspberryPi

Esempi di uso del VirtualRobot¶

progetto virtualrobotusage

  • bw24analisi0.qak

Modello PRELIMINARE dell’analisi del problema BoundaryWalk24.

  • bw24naive.qak

Implementazione di una soluzione al problema BoundaryWalk24, che interagisce direttamente con il Il VirtualRobot.

  • VrobotLLMoves24.java

POJO di supporto per la interazione di un actorqak con Il VirtualRobot

  • bw24proto1.qak

Un prototipo che copre i requisiti BW-Req1 e BW-Req2 del problema BoundaryWalk24 usando VrobotLLMoves24

  • bw24sonarwolfmock.qak

Un simulatore del sonar su RaspberryPi che emette via MQTT eventi percepibili da bw24proto1all.qak.

  • vrqak24.qak

Un servizio per permette l’uso remoto del VirtualRobot23

  • vrqaktest.qak

Un test sulle operazioni di VrobotLLMoves24

  • bw24proto2.qak

Un prototipo che copre tutti i requisiti del problema BoundaryWalk24 usando Vrqak24

Execution tasks¶

Per eseguire vrqak24.qak:

task runVr(type: JavaExec) {
    group = "Execution"
    description = "Run the main class with JavaExecTask"
    classpath = sourceSets.main.runtimeClasspath
    main = 'it.unibo.ctxvrqak.MainCtxvrqakKt'
}

Per eseguire bw24sonarwolfmock.qak :

task runWolf(type: JavaExec) {
    group = "Execution"
    description = "Run the main class with JavaExecTask"
    classpath = sourceSets.main.runtimeClasspath
    main = 'it.unibo.ctxwolf.MainCtxwolfKt'
}

Il sonar su RaspberryPi¶

  • Il simulatore bw24sonarwolfmock.qak fa sistema con a bw24proto1all.qak.

  • Il sonar su RaspberryPi che fa sistema con a bw24proto1all.qak è sonarbw24 che emette l’evento Event wolf:wolf(D)

Oltre il servizio vrqak24

basicrobot¶

Il servizio Vrqak24 è stato introdotto per permettere l’uso remoto del VirtualRobot23.

Questo servizio può costituire il punto di partenza per la realizzazione di un servizio più esteso, che includa anche la gestione di uno specifico robot fisico. (d’ora in poi denominato basicrobot)

In via preliminare, questo nuovo servizio si può impostare come mostrato nel modello basicrobot.qak.
